Python换源#

1. 临时换源#

可以在使用pip的时候在后面加上-i参数，指定pip源

pip install scrapy -i https://pypi.tuna.tsinghua.edu.cn/simple

2. 永久换源#

永久修改： linux: 修改 ~/.pip/pip.conf (没有就创建一个)， 内容如下：

[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
