    删除WordPress

    删除WordPress 删除网络文件： rm -Rf /var/www/html/* 删除数据库。首先获取mysql的root密码（通过ssh登录时显示在MOTD中）。 mysql -uroot -p 输入密码后使用语句： DROP DATABASE wordpress; 删除WordPress数据库 exit;

    安装完virtualbox后，提示内核问题

    安装完virtualbox后，提示内核问题 解决方法 安装内核匹配版本 sudo pacman -S linux419-virtualbox-host-modules 重新加载内核模块 sudo /sbin/rcvboxdrv 然后重启即可
